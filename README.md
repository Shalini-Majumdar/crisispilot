# CrisisPilot

CrisisPilot is an AI Failure Simulator and Incident Commander.

It helps developers test AI application failures in a safe environment by simulating incidents such as latency spikes, tool failures, and token-cost explosions.

The system combines:

- Google Gemini
- Google Cloud Agent Builder
- Dynatrace Observability
- OpenTelemetry

Features:

- Failure Simulation
- Incident Investigation
- Root Cause Analysis
- Human Approval Workflow
- AI Generated Postmortems
